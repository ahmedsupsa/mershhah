# حالة المشروع الحالية - منصة مرشح (Mershhah)

## 1. نظرة عامة
مرشح هو **الواجهة الرقمية الموحدة** للمطاعم والمقاهي، تعمل داخل الفرع عبر QR Code وخارجه كرابط ذكي. المنصة مبنية على ركيزتين:
1.  **واجهة عميل ذكية (بديل Linktree):** تعمل كمدخل للبيانات من خلال تجميع كافة روابط المشروع (منيو، عروض، فروع، إلخ) في صفحة واحدة.
2.  **منصة أدوات داخلية:** "صندوق أدوات" لصاحب المشروع يحتوي على أدوات تحليلية وإبداعية (مدعومة بالذكاء الاصطناعي) لتحليل البيانات المجمعة وتحويلها إلى رؤى لزيادة الأرباح.

## 2. الهيكل التقني (Tech Stack)
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Shadcn/UI
- **Database & Auth:** Firebase (Firestore, Authentication, Storage)
- **AI:** Google Genkit + Gemini
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Language:** Arabic (RTL support)

## 3. تقسيم النظام (Roles & Dashboards)

### أ- لوحة تحكم المدير العام (Admin Dashboard)
- **إدارة المشتركين:** تفعيل الحسابات والاشتراكات يدويًا، إدارة الصلاحيات، ومتابعة الأداء.
- **القسم المالي:** تتبع الفواتير والاشتراكات.
- **إدارة المتجر:** التحكم في الأدوات والخدمات المتاحة في "متجر الأدوات".
- **الدعم الفني:** نظام محادثات مباشرة مع أصحاب المشاريع.

### ب- لوحة تحكم صاحب المشروع (Owner Dashboard)
- **الإحصائيات:** نظرة سريعة على تفاعل العملاء.
- **إدارة المحتوى:** التحكم في المنيو، العروض، والفروع.
- **متجر الأدوات:** تفعيل أدوات إضافية.
- **الاستوديو:** تخصيص الهوية البصرية وإنشاء صور إعلانية.

### ج- واجهة العميل (Customer Interface)
- **الواجهة الموحدة (`/hub`):** صفحة هبوط لكل مشروع تجمع كل الروابط والمعلومات.
- **المنيو الرقمي:** تصفح الأصناف بشكل جذاب.
- **المساعد الذكي:** بوت محادثة للرد على استفسارات العملاء.

## 4. المجلدات الرئيسية
- `src/app`: تحتوي على الصفحات والمسارات (Admin, Owner, Auth, Public).
- `src/components`: المكونات القابلة لإعادة الاستخدام (UI, Shared, Dashboard).
- `src/ai`: منطق الذكاء الاصطناعي (Genkit Flows).
- `src/lib`: الأدوات المساعدة، أنواع البيانات، وإعدادات Firebase.
- `src/hooks`: React Hooks مخصصة (مثل `useUser`).
