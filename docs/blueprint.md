# **App Name**: مرشح

## Core Features:

- Restaurant Management: Allows admin to manage restaurants, users, and applications within the system.
- Menu Management: Enables restaurant owners to manage menus including items, descriptions, and pricing.
- AI-Powered Chat Assistant: Implements Genkit AI with Gemini 1.5 model to provide smart and relevant responses to customer inquiries, leveraging a knowledge base built from the restaurant's data. The AI tool helps respond to client requests.
- Anonymized Message Analytics: Analyzes anonymized chat messages (text and timestamp only) to identify frequent topics, customer satisfaction trends, and peak hours, displayed on dashboards for admins and restaurant owners.
- Role-Based Access Control: Implements Firebase Authentication for different user roles (Admin, Restaurant Owner, Customer) with email/password login. Each role has access to specific dashboards and features.
- Offer Management: Allows restaurant owners to create and manage promotional offers.
- Media Asset Management: Integrates Cloudinary for storing and serving images such as restaurant logos, menu items, and banners.

## Style Guidelines:

- Primary color: Deep Emerald (#0A9396) for a sophisticated and inviting feel, inspired by rich Middle Eastern cuisine.
- Background color: Soft Sand (#F4F4F4) to create a clean and welcoming environment.
- Accent color: Golden Amber (#D90429) for highlighting key elements, evoking warmth and spice.
- Headline font: 'Belleza' (sans-serif) for an elegant and modern feel; Body font: 'Alegreya' (serif) for readability in longer texts.
- Use clean and modern icons relevant to the restaurant industry (food items, cutlery, location, etc.).
- Design a responsive layout that adapts to different screen sizes (desktop, tablet, mobile) while maintaining a consistent look and feel. Implement right-to-left (RTL) support for the Arabic interface.
- Incorporate subtle animations and transitions to enhance user experience and provide visual feedback.
